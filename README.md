# SQL Projects Portfolio

A collection of PostgreSQL database projects created while learning database design, SQL querying, and relational modeling.

## Projects

### 01. Library Management System

A database for managing books, authors, members, and borrowing records.

**Concepts**

* Primary Keys
* Foreign Keys
* JOINs
* One-to-Many Relationships

### 02. E-Commerce Database

A database for managing customers, products, orders, and order items.

**Concepts**

* Aggregations
* Revenue Reports
* Customer Orders
* Product Management

### 03. University Management System

A database for managing students, professors, courses, and enrollments.

**Concepts**

* Many-to-Many Relationships
* Composite Keys
* Enrollment Tracking

### 04. Social Media Database

A database that simulates a social media platform.

**Concepts**

* Self Joins
* Likes and Comments
* Social Relationships

### 05. Sales Analytics Database

A database for analyzing sales and generating reports.

**Concepts**

* GROUP BY
* Aggregations
* Reporting
* Analytics
