# 📊 Sales Analytics Database

A PostgreSQL database project focused on sales reporting and business analytics.

## 📌 Project Overview

This project simulates a company's sales system and demonstrates how SQL can be used to generate business insights.

The database tracks:

* Products
* Categories
* Sales Transactions

Using SQL queries, we can answer business questions such as:

* Which products generate the most revenue?
* Which category sells the most?
* What is the monthly revenue?
* What are the top-selling products?

---

## 🗄️ Database Schema

### Tables

| Table      | Description         |
| ---------- | ------------------- |
| categories | Product categories  |
| products   | Product information |
| sales      | Sales transactions  |
